# Contributing

Please follow these rules when contributing to **awesome-pypi-packages**:

* Add one link per pull request.
* Make sure the pull request title is in the format: `Add project-name`.
* Write down the reason why the library is useful.
* Add the link in this format:
  `- [project-name](https://example.com/#readme) - A short description that ends with a period.`
* Keep descriptions concise and short.
* Add a new section if needed.
* Add a short description for the section.
* Add the section title to the Contents list.
* Search previous pull requests or issues before making a new one to avoid duplicates.
* Do not mention Python in the description, as it is implied.
* Check your spelling and grammar.
* Remove any trailing whitespace.
* Ensure the project is actively maintained.
* Place the entry at the bottom of the correct section.
* Run `awesome-lint` before submitting.

By submitting a contribution, you agree that it follows these rules and fits the purpose of this list.
